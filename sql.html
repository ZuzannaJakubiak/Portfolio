Data Cleaning

In this project, I applied SQL techniques to perform comprehensive data cleaning. This included creating backup tables to safeguard the original data, removing duplicates using advanced window functions and CTEs, and standardizing various fields for consistency. I addressed null and blank values by updating missing entries and replacing empty fields as needed. Finally, I refined the dataset by removing unnecessary columns and rows with missing critical data, ensuring a clean and accurate dataset ready for analysis.

Layoffs Trends Exploratory Data Analysis

This project involves a detailed analysis of layoff data to identify and understand trends over time. Using SQL queries, it examines various aspects of layoffs, including the total number of employees laid off, the percentage of workforce affected, and the financial context of these layoffs. The analysis covers multiple dimensions, such as company-specific layoffs, geographic distribution by country, and annual and monthly trends. It also tracks cumulative layoffs to observe how the total number of layoffs evolves over time. By ranking companies based on their annual layoffs, the project highlights the most significantly impacted organizations. This comprehensive approach provides valuable insights into the factors influencing layoffs and helps identify key patterns and trends across different sectors and regions.

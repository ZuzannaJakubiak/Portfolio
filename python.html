Backpropagation

In this project, I implemented the Backpropagation algorithm for a binary classification problem using Python. The project involved generating a synthetic dataset with three input variables to classify data points as either yellow or purple dots. I utilized libraries such as NumPy and Matplotlib for data manipulation and visualization.
The model was built from scratch, featuring weight initialization, forward propagation, and cost function calculation. I applied various activation functions, including sigmoid and leaky ReLU, and implemented gradient descent to optimize the model parameters. The training process involved iterating over the dataset multiple times to minimize the loss and enhance accuracy.
Finally, I evaluated the model's performance by calculating its accuracy on both training and test data, visualizing the results to showcase the model's predictive capabilities.

Next Day Market - Time Series Prediction

In this project, I developed a neural network model to predict electricity prices on the Polish Next Day Market 24 hours in advance. The dataset comprised key features, including the target price per megawatt-hour (z≈Ç/MWh), trading volume, hour, and date.
I utilized libraries such as Pandas, NumPy, Matplotlib, Seaborn, and TensorFlow to preprocess the data, visualize trends, and build the neural network. The data was formatted and indexed by date and hour, allowing for effective time series analysis.
The neural network architecture consisted of several dense layers, employing the ReLU activation function to enhance model performance. I compiled the model using the Adam optimizer and mean squared error loss function, training it over multiple epochs to minimize prediction errors.
After training, I evaluated the model's predictions against the actual values, visualizing the results to demonstrate the model's accuracy. Finally, the predicted values were saved for further analysis.

Cat or Dog? - Image Classification

In this project, I developed a Convolutional Neural Network (CNN) model to classify images of cats and dogs. The dataset was organized into training, validation, and test sets.
Key features included binary labels indicating whether an image contained a cat or a dog. I utilized libraries such as TensorFlow, NumPy, Matplotlib, and OpenCV for data preprocessing, visualization, and model development. Image augmentation and rescaling were applied to enhance the model's robustness.
The CNN architecture comprised multiple convolutional and max-pooling layers, using the ReLU activation function to improve feature extraction. The final layer employed a sigmoid activation for binary classification.
After training, I evaluated the model's accuracy on the validation set and generated predictions for the test dataset. The results were compiled into a submission file, and the training progress was visualized to demonstrate the model's learning effectiveness.

Amazon Price Tracker - Webscrapping

In this project, I developed a web scraper to track the price of a specific product on Amazon. Using the BeautifulSoup and Requests libraries, I extracted key information, including the product title and price, from the Amazon product page.
The scraper sends a request to the specified URL, retrieves the HTML content, and parses it to extract the relevant data. The product title and price are then recorded along with the current date. I implemented functionality to append this data to a CSV file for ongoing tracking.
To automate the process, I set up a loop that checks the product price at regular intervals. The collected data can be analyzed later to observe price trends over time.
